================================================================================
  🔧 PYTHON 3.13 SPECIFIC FIX
================================================================================

PROBLEM: pydantic 2.5.0 doesn't have pre-built wheels for Python 3.13
         This forces pip to compile from source, which requires Rust compiler

SOLUTION: Use pydantic 2.9+ which has Python 3.13 wheels (no compilation)

================================================================================
  ✅ CORRECT INSTALL COMMAND FOR PYTHON 3.13
================================================================================

pip3.13 install "pydantic>=2.9.0,<3.0.0"

This will:
✓ Download pre-built wheel (no Rust needed)
✓ Install in seconds instead of minutes
✓ Avoid "Disk quota exceeded" error
✓ Work with your models_v3_pythonanywhere.py

================================================================================
  📦 FULL INSTALLATION SEQUENCE
================================================================================

# 1. Clean first
rm -rf ~/.cache/puccinialin
pip3.13 cache purge

# 2. Install packages one by one (easier to debug)
cd ~/Concierge-Analyzer/mysql_api

pip3.13 install Flask==2.3.3
pip3.13 install Flask-CORS==4.0.0
pip3.13 install mysql-connector-python==8.2.0
pip3.13 install python-dotenv==1.0.0
pip3.13 install jsonschema==4.19.2
pip3.13 install python-dateutil==2.8.2
pip3.13 install "pydantic>=2.9.0,<3.0.0"  # ← CRITICAL: Use 2.9+

# 3. Verify
pip3.13 list | grep pydantic
# Should show: pydantic  2.9.x (or 2.10, 2.11, etc - all compatible)

================================================================================
  🔄 OR USE requirements_pythonanywhere.txt
================================================================================

# This file has been updated with correct pydantic version
pip3.13 install -r requirements_pythonanywhere.txt

================================================================================
  ⚠️ WHY VERSION MATTERS
================================================================================

Python 3.13 support by pydantic version:

pydantic 2.5.0  → ❌ No wheel, needs Rust compilation
pydantic 2.6.0  → ❌ No wheel, needs Rust compilation  
pydantic 2.7.0  → ❌ No wheel, needs Rust compilation
pydantic 2.8.0  → ❌ No wheel, needs Rust compilation
pydantic 2.9.0+ → ✅ Pre-built wheel available!
