Your .git folder was deleted. Here's how to restore it:

================================================================================
  OPTION 1: RE-CLONE FROM GITHUB (RECOMMENDED)
================================================================================

# 1. Go to parent directory
cd ~

# 2. Backup current folder (just in case)
mv Concierge-Analyzer Concierge-Analyzer-backup

# 3. Clone fresh from GitHub
git clone https://github.com/wsmontes/Concierge-Analyzer.git

# 4. Done! Now you can git pull normally
cd Concierge-Analyzer
git pull origin main

# 5. Install packages
cd mysql_api
pip3.13 install -r requirements.txt

# 6. Reload web app

================================================================================
  OPTION 2: REINITIALIZE GIT IN EXISTING FOLDER
================================================================================

If you want to keep your current files:

# 1. Go to project root
cd ~/Concierge-Analyzer

# 2. Initialize git
git init

# 3. Add remote
git remote add origin https://github.com/wsmontes/Concierge-Analyzer.git

# 4. Fetch from GitHub
git fetch origin main

# 5. Reset to match GitHub
git reset --hard origin/main

# 6. Set tracking
git branch --set-upstream-to=origin/main main

# 7. Now git pull will work
git pull origin main

================================================================================
  OPTION 3: MANUAL FILE UPLOAD (IF GIT ISSUES PERSIST)
================================================================================

If git continues to have problems:

1. Just upload the updated files via PythonAnywhere Files tab:
   - mysql_api/models_v3.py
   - mysql_api/requirements.txt
   - mysql_api/wsgi_v3.py

2. Install packages:
   cd ~/Concierge-Analyzer/mysql_api
   pip3.13 install -r requirements.txt

3. Reload web app
